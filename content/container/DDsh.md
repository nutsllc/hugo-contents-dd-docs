``DDsh`` コマンドは 稼働中の Docker コンテナの中に入ります。

コンテナ内ではシェル(``sh``)を用います。

``DDsh`` コマンドを実行すると peco が起動し稼働中のコンテナ一覧が表示されるので、その中から、中に入りたいコンテナを選択します。

``DDsh`` コマンドは ``docker exec -it <コンテナID> /bin/sh`` と同等です。

``exit`` コマンドでコンテナから抜けることが出来ます。

```bash
$ DDsh
```
